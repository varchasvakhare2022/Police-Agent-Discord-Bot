# Police-Agent-Discord-Bot
Moderation + Utility Discord Bot

## How to run the bot
- Clone this repository `git clone https://github.com/varchasvakhare2022/Police-Agent-Discord-Bot`
- Edit `example.env` to `.env` and put your bot's token there
- Open cmd and run `pip install -r requirements.txt`
- Now edit certain values in the code like `server_id`, `channel_id`, `role_name` etc
- Now run `python3 __main__.py`
You are all set now!
